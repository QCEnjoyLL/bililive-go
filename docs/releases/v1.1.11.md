# v1.1.11 更新说明

本版本主要修复更新说明展示和直播间列表的小体验问题，同时完善后续发布流程。

## 改进内容

- 修复直播间列表操作栏中“配置”按钮字号偏小的问题，现在它与“停止监控”“删除”“文件”等操作项保持一致。
- 发布流程新增中文更新说明文件校验。以后每次发布新版本前，都需要在 `docs/releases/vX.Y.Z.md` 写好对应版本的中文更新说明；缺少说明文件时，GitHub Release 发布会直接失败，避免再出现空 Release 正文。
- GitHub Release 发布时会自动使用上述中文更新说明作为 Release 正文，更新页面不再只能看到固定的三条默认说明。
- 程序检查更新时，如果 GitHub API 返回的 Release 正文为空，或因限流只能走备用更新路径，会尝试从仓库内读取同版本的中文更新说明并展示。

## 升级建议

- 建议从 v1.1.10 升级到 v1.1.11，以获得更准确的更新日志展示。
- 本版本不改变录制行为、更新策略或现有配置项。
