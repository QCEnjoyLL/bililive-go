# v1.2.2 更新说明

本版本是一次 WebUI 登录与更新体验修复，重点解决开启鉴权后登录页图标无法显示，以及程序更新完成后因为需要重新登录而停留在更新等待页的问题。

## 登录页资源修复

- 修复开启 WebUI 鉴权后登录页图标裂开的问题。
- `/favicon.ico` 和 `/manifest.json` 现在会作为公开资源放行，不会再被登录鉴权拦截。
- 针对当前图标文件“扩展名是 `.ico`、实际内容是 PNG”的情况，服务端会显式按 `image/png` 返回，避免浏览器误判导致图标加载失败。

## 更新后自动恢复

- 优化更新页面的重启等待逻辑，避免更新完成后一直停留在“等待服务器重启”的页面。
- 更新页现在会持续检测 `/api/info`：
  - 如果新版本启动后会话仍有效，会自动刷新页面。
  - 如果新版本启动后登录会话失效，检测到 `401/403` 后会自动跳转到登录页。
  - 登录地址会带上 `next` 参数，重新登录后可以回到原来的更新页面。
- 增加轮询保护，避免重复创建多个重启检测定时器。
