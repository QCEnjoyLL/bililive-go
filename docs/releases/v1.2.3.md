# v1.2.3 更新说明

本版本是一次 WebUI 深色主题显示修复，重点处理部分页面在深色主题或不同配色下仍然出现黑字、白底、对比度不一致的问题。

## 登录页面

- 修复深色主题下登录页标题文字没有随主题变化的问题。
- `BiliLive-go`、`进入 WebUI`、`欢迎回来` 现在会统一使用当前主题的主文字颜色，不会再在深色背景上显示成黑色。

## 更新页面

- 修复更新页面和顶部更新弹窗里的「更新日志」区域仍然使用浅色白底的问题。
- 更新日志内容现在会跟随当前主题使用合适的背景、边框、文字颜色和滚动条样式。
- 优化更新日志区域的代码/日志类文本显示，提升深色主题下的可读性。

## 直播间列表

- 修复深色主题下直播间名称不够明显的问题。
- 将直播间名称链接、列表标题和表格文字接入 WebUI 主题变量，减少硬编码浅色文字导致的显示异常。

## 验证

- 已通过前端构建检查。
- 已通过 Go 服务端相关测试。
- 已通过 Go vet 静态检查。
